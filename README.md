# 📊 Fitness Attendance Analysis (Age Group Focus)

This project uses fake gym attendance and member data to show how different age groups (Under 40 vs 40 and Over) participate in group fitness classes. It's designed to help health and fitness businesses make better decisions with data to scale their business.

## 💡 Goals
- Compare attendance between age groups
- Visualize which age group prefers which class type
- Help marketers better target each demographic

## 🧰 Tools
- SQL (data queries)
- Python (Pandas, Matplotlib)
- Google Colab (Notebook)
- Google Sheets (fake data creation)

## 📈 Sample Insights
- People under 40 made up 60% of total attendance
- Zumba and Spin were most popular for 40+
- CrossFit had more consistent younger attendance
